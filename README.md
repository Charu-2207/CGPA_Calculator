# CGPA Analytics Platform

A full-stack CGPA calculator and academic analytics platform for students to track, calculate, and visualize semester-wise academic performance.

## Features

- Student authentication
- Semester-wise subject management
- SGPA & CGPA calculation
- Dynamic subject loading
- Arrear/elective subject support
- Interactive analytics dashboard
- Performance visualization
- Responsive modern UI

## Tech Stack

- React.js
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- Recharts
- Vercel
